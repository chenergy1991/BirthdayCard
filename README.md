SpringBoot是轻量级的Java开发框架，在继承Spring框架原有优秀特性的同时，通过简化配置来进一步简化了Spring应用的搭建和开发过程。

小编在近期学习了最基础的Spring Boot开发，期间编写了这个小Demo。小编希望通过维护该开源项目的代码和博客，以提高自己“安全开发”的能力。


小Demo的功能很简单（页面也很粗糙），主要为以下几点：

（1）通过网页对人物生日信息的增删改查

（2）简易的管理员登录功能（未登录则不能访问页面）

（3）生日提醒功能

（4）生日信息Excel的导入、导出

注：代码中会存在一些问题，比如：

（1）暂未对一些漏洞进行防御

（2）暂未对参数进行校验

（3）功能不够完善
